# RESULT.md 模板 — 改修总结

> 本文件是 `code-it` 技能产出的 `RESULT.md` 章节结构与填写规范。
> 模型在生成 `RESULT.md` 时应严格按本结构组织,作为本次任务开发工作的最终交付物。
>
> **版本感知**:本文件存放在 `./assistants/<版本号>/code/<任务编码>/RESULT.md`,文档头应明确记录版本号。

---

## 文档头
```

- 任务来源:./assistants/<版本号>/plan/<需求编号>/PLAN.md §3
- 详细设计:./assistants/<版本号>/plan/<需求编号>/RESULT.md §X

- 创建:<YYYY-MM-DD HH:mm>

```
