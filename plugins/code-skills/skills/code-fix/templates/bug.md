### 状态枚举
- `报告`:刚被登记,尚未开始调查
- `调查中`:正在分析根因
- `修复规划中`:已调 `code-plan` 产出 `fix-plan.md`
- `修复编码中`:已调 `code-it` 实施修复
- `已修复-待验证`:代码已改完,等待验证
- `已修复-已验证`:测试通过 / 人工确认
- `已关闭-非缺陷`:调查后确认不是 bug
- `已关闭-不修复`:确认是 bug 但决定不修
- `已取消`:不再跟进
- `阻塞`:等待外部因素
