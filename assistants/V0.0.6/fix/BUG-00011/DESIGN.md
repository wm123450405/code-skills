## 5. 关键流程

### 5.1 清理流程(每个文件一致)

```
读取文件 → 定位 "## 不要做的事" 板块起点 → 定位板块结束位置
  → 判断:若板块是文件末尾,删除至上一 `---` 行(含)
  → 判断:若板块非文件末尾,删除该板块所有行(含标题 + 空行 + 占位 + 上一 `---` 后的 1 空行)
→ 用 Edit 单次精准替换
→ 验证文件可读 + 行数减少 + 无残留 "## 不要做的事"
```

### 5.2 验证流程

每个文件清理后,执行:
1. `Read` 文件,搜索 `## 不要做的事` → 应为 0 命中
2. `Read` 文件,搜索 `本节保留位置` → 应为 0 命中
3. 文件仍以有效内容结尾(非空)

## 6. 方案选型

### 6.1 方案 A:整段删除(选定)

- **做法**:直接 Edit 删除整段板块(标题 + 空行 + 占位行)
- **优点**:彻底、无残留、信息密度提升
- **缺点**:若未来需要"反向必须不做"清单,需重新添加
- **权衡**:用户已确认,且 REQ-OPT-00002 整改后实际无需要

- 保留板块标题,删除占位行 → 板块下为空,仍显噪音

### 6.3 方案 C(已否决)
