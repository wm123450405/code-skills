## 偏离 1:`doc-conventions.md:113` 正面示例含旧串,本任务未修复
- **类别**:代码偏离规范(规范文件本身含过期字面量)
- **依据**:
  - 设计要求:详细设计 §3.4 默认 CLAUDE.md 0 变更;同时 PLAN.md §2.3 仅核查 CLAUDE.md,未提及规范文件

  - 长期:用户按"正面示例"复制 README 时,会复制过期的 install 命令(误导风险,**中**)
